# aghdam.nl

## Create and deploy a new post

### 1. Create a new blog post
```sh
hugo new post/{KEBAB_CASE_BLOG_NAME}.md
```

### 2. Commit new changes
```sh
git add -A
git commit -m "{COMMIT_MESSAGE}"
```

### 3. Update dist folder
```sh
huge
```

### 4. Commit and push
```sh
git add -A
git commit -m "deploy"
git push
```
